# Withlive Backend Server

## Quick Start

1. YOLOv5 Repository 복사

```bash
git clone https://github.com/ultralytics/yolov5.git
```

2. touch 모델 준비. `(./models/)`

3. venv Activate

```bash
python -m venv .venv && . .venv/bin/activate
```

4. Python Package 설치

```bash
pip install -r requirements.txt
```

5. Server Start

```bash
python3 app.py
```
